Javascript Project Personal #1 - Find the Unique String of Array

Description

Terdapat sebuah array yang memiliki beberapa nilai. Nilai tersebut berisi string harus semua. Setiap nilai dalam array tersebut terdapat satu string yang memiliki karakter yang tidak dimiliki oleh string lainnyaa.

Pengujian

Buka console pada developer tools pada browser menggunakan shortcuts dengan menekan Command + Option + J (Mac) atau Control + Shift + J (Windows, Linux, ChromeOS)

Tulis function findUniqueString(array). Parameter pertama array masukkan nilai array yang akan dicari nilai unik nya. Dalam satu array hanya bisa berisi satu tipe data saja yaitu string.

Contoh sebagai berikut

  • findUniqueString(["abc", "acb", "bac", "foo", "bca", "cab", "cba"]) // foo
  • findUniqueString( ["Tom Marvolo Riddle", "I am Lord Voldemort", "Harry Potter"]) // Harry Potter
  • findUniqueString( ["Aa", "aaa", "aaaaa", "BbBb", "Aaaa", "AaAaAa", "a"]) // BbBb
  • findUniqueString([" ", "a", " "]) // a
