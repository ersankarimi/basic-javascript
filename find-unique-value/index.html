Javascript Project Personal #1 - Find the Uniqe Number or String

Description

Terdapat sebuah array yang memiliki beberapa nilai. Nilai tersebut bisa berisi string atau number semua. Setiap nilai dalam array tersebut terdapat satu nilai yang berbeda (unik) dari nilai lainnya.

Pengujian

Buka console pada developer tools pada browser menggunakan shortcuts dengan menekan Command + Option + J (Mac) atau Control + Shift + J (Windows, Linux, ChromeOS)

Tulis function findUniqValue(array, type). Parameter pertama array masukkan nilai array yang akan dicari nilai unik nya dan parameter kedua type masukkan nilai tipe data pada array tersebut (string). Dalam satu array hanya bisa berisi satu tipe data saja string atau number.

Contoh sebagai berikut

  • findUniqValue([1, 0, 0], "number") // 1
  • findUniqValue([3, 10, 3, 3, 3], "number") // 10
  • findUniqValue([1, 1, 2, 1, 1], "number") // 2
  • findUniqValue(["abc", "acb", "bac", "foo", "bca", "cab", "cba"], "string") // foo
  • findUniqValue( ["Tom Marvolo Riddle", "I am Lord Voldemort", "Harry Potter"], "string") // Harry Potter
  • findUniqValue( ["Aa", "aaa", "aaaaa", "BbBb", "Aaaa", "AaAaAa", "a"], "string") // BbBb
  • findUniqValue([" ", "a", " "], "string") // a
